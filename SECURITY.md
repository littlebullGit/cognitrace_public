# Security Policy

## Supported Versions

Currently, only the latest release of CogniTrace is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Please report security issues by opening an issue or contacting the maintainers directly. Do not disclose the vulnerability publicly until it has been addressed. We aim to review and respond to reports within 48 hours.

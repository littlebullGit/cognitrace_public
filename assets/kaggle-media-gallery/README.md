# Kaggle Media Gallery Upload Set

Recommended upload order for the Gemma 4 Good Hackathon writeup media gallery.
Kaggle's media upload flow currently asks for an item title only, not a caption, so the suggested text below is written as short upload titles.
Kaggle also uses a landscape preview crop, so prefer the `1280x720` composite files below instead of raw vertical iPhone screenshots.

| File | Purpose | Suggested title |
| --- | --- | --- |
| `00-card-preview-1280x720.png` | Replacement card/thumbnail image sized for Kaggle's 16:9 preview crop. | CogniTrace on-device screening |
| `01-combined-screening-flow-1280x720.png` | Combines onboarding, voice task, and result explanation. | Screening flow |
| `02-combined-offline-transparency-1280x720.png` | Combines offline model download, biomarker details, and low-risk result. | Offline and transparent |
| `03-combined-multilingual-handoff-1280x720.png` | Combines multilingual Ask Gemma, doctor discussion guide, and elevated result. | Multilingual handoff |

Use `00-card-preview-1280x720.png` instead of the older `00-card-thumbnail-1120x560.png`; the older image is 2:1 and Kaggle trims the sides in the preview crop.
Use the three `combined-*` files instead of the raw `01`-`07` portrait screenshots for the same reason.

Skip `assets/iphone-screenshots/IMG_0012.PNG` for the Kaggle gallery; it is just the microphone permission prompt and does not strengthen the pitch.
